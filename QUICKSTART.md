# 🚀 Quick Start Guide - Day 1

## Get Running in 5 Minutes

### 1. Install Dependencies (2 minutes)

```bash
# Install required packages
pip install numpy python-dotenv supabase

# Or use requirements file
pip install -r requirements.txt
```

### 2. Start Learning (Now!)

```bash
# Run the files in order
python3 day1_01_introduction.py    # 30 min - Theory
python3 day1_02_numpy_basics.py     # 45 min - Arrays
python3 day1_03_numpy_advanced.py   # 60 min - Operations
python3 day1_04_similarity.py       # 45 min - Recommendations
```

### 3. For BingeItBro Project (Optional for today)

Only needed for the final project file:

```bash
# Copy environment template
cp .env.example .env

# Edit .env and add your Supabase credentials
# (Find them in your Supabase dashboard > Settings > API)
nano .env  # or use any text editor

# Run the project
python3 day1_06_bingeitbro_project.py
```
